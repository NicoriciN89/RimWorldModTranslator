RimWorld Mod Translator
=======================

Локальный офлайн-переводчик модов RimWorld. Не отправляет ваши файлы никуда
в интернет — весь перевод происходит на вашем компьютере.

КАК ЗАПУСТИТЬ
-------------
Зайдите в папку RimWorldModTranslator и дважды кликните
RimWorldModTranslator.exe. Python устанавливать не нужно. Не переносите сам
exe отдельно из этой папки — рядом с ним обязательно должна быть папка
_internal, там все нужные библиотеки. Если хотите переместить программу —
переносите целиком папку RimWorldModTranslator, а не только exe из неё.

Windows SmartScreen может показать предупреждение "Windows защитила ваш
компьютер", потому что exe не подписан цифровой подписью издателя (это
самостоятельно собранная утилита, а не программа известного разработчика).
Чтобы запустить: нажмите "Подробнее" -> "Выполнить в любом случае".

ЕСЛИ АНТИВИРУС РУГАЕТСЯ ("ЭТО ВИРУС?")
----------------------------------------
Программа собрана стандартным открытым инструментом PyInstaller (упаковывает
Python-код в exe) — тем же способом, что и множество легитимных утилит.
Некоторые антивирусы иногда эвристически помечают такие сборки как
подозрительные просто по паттерну упаковки, а не потому что нашли реальный
вредоносный код (так называемый false positive — ложное срабатывание).
В программе вообще нет сетевого кода для перевода — русский, украинский,
немецкий и французский встроены в неё, для них ничего не скачивается.
Опциональная LLM-доработка обращается только к Ollama на вашем же
компьютере (localhost), которую вы сами устанавливаете и запускаете —
сама программа в интернет не выходит.

Более ранняя сборка проверялась на VirusTotal (сервис, прогоняющий файл
через ~70 антивирусов сразу): 2 из 68 движков дали срабатывание (Bkav Pro,
SecureAge) — оба известны повышенной эвристической чувствительностью именно
к PyInstaller-сборкам, без конкретного названия угрозы (типичный признак
ложного срабатывания). Все основные антивирусы (Kaspersky, ESET,
BitDefender, Avast, Windows Defender, CrowdStrike, Google и др.) файл не
отметили. У каждой новой версии хеш файла другой — при желании проверьте
свой exe сами на https://www.virustotal.com (загрузите файл и посмотрите
вердикт десятков антивирусов сразу). Если ваш антивирус заблокировал файл —
добавьте папку с программой в исключения антивируса.

КАК ПОЛЬЗОВАТЬСЯ
-----------------
1. Кнопка "Обзор..." рядом с "Папка мода" — выберите папку мода RimWorld
   (там, где лежит папка About с файлом About.xml).
2. Выберите язык перевода из списка. Программа полностью офлайн и ничего
   не скачивает из интернета: русский, украинский, немецкий и французский
   встроены в неё и работают сразу. Для любого другого языка: скачайте его
   из релиза "language-packs" на странице проекта на GitHub (там есть
   испанский, португальский, португальский (Бразилия), итальянский,
   польский, турецкий, китайский, японский, корейский — а любой другой язык
   можно взять на https://www.argosopentech.com/argospm/index/), распакуйте
   zip и скопируйте получившуюся папку в папку `_internal\bundled_packages`
   рядом с RimWorldModTranslator.exe. Перезапустите программу — новый язык
   появится в списке.
3. Движок перевода:
   - "Только Argos (быстро)" — обычный офлайн-перевод, за секунды. Работает
     сразу, ничего дополнительно ставить не нужно.
   - "Argos + LLM-доработка" и "Только LLM" — делают перевод более грамотным
     (лучше падежи/род), но намного медленнее (секунды на КАЖДУЮ строку).
     Для этого режима одной установки Ollama НЕДОСТАТОЧНО — нужно ещё
     скачать саму языковую модель (qwen2.5:7b) — она открытая (open source)
     и полностью бесплатная, никаких подписок или оплаты не требует. Как и
     локальный запуск моделей на своём ПК через Ollama — тоже полностью
     бесплатен и без ограничений (платные тарифы на сайте ollama.com
     касаются только ИХ облачного сервиса — нам он не нужен):
       а) скачайте и установите Ollama: https://ollama.com/download
       б) откройте командную строку (Win+R, введите cmd, Enter) и выполните:
          ollama pull qwen2.5:7b
          (скачает ~4.7 ГБ, один раз, нужен интернет только на этом шаге —
          сама модель бесплатная, платите только за свой интернет-трафик)
       в) после этого модель появится в выпадающем списке "Модель Ollama"
          в программе (или нажмите там "Обновить список")
     Если не хотите возиться с Ollama — просто используйте "Только Argos".
4. Нажмите "Перевести". Готовый мод-русификатор появится в указанной папке
   вывода (по умолчанию — папка output рядом с exe).
5. Готовую папку с переводом устанавливайте как обычный мод RimWorld —
   рядом с оригинальным модом, ниже него в списке модов.

ОЧЕРЕДЬ МОДОВ (ПЕРЕВОД НЕСКОЛЬКИХ МОДОВ ЗА РАЗ)
------------------------------------------------
Кнопкой "Добавить" соберите в список несколько папок модов и нажмите
"Перевести" — они обработаются по очереди. Одинаковые строки в разных модах
переводятся один раз (общая память переводов), ошибка одного мода не
прерывает остальные. Если очередь пуста, переводится мод из поля "Папка
мода", как обычно.

Запущенный перевод можно остановить кнопкой "Отмена" — частично
переведённый результат останется на диске.

РУЧНЫЕ ПРАВКИ ПЕРЕВОДА
------------------------
Кривую строку можно исправить прямо в файле готового перевода (XML/txt).
При следующей генерации программа сама заметит вашу правку, сохранит её в
файл manual_overrides.json рядом с переводом и с тех пор всегда будет
подставлять ваш вариант — даже при полном повторном переводе. Чтобы
отменить правку — удалите её строку из manual_overrides.json.

ПОВТОРНЫЙ ПЕРЕВОД / ОБНОВЛЕНИЕ МОДА
-------------------------------------
Если мод обновился и вы хотите доперевести только новые строки, а не всё
заново — поставьте галочку "Режим обновления" и укажите ту же папку вывода,
куда переводили в прошлый раз. Уже переведённые и не изменившиеся строки
(включая ваши ручные правки) останутся нетронутыми.

Настройки окна (папки, язык, движок, галочки) сохраняются между запусками
автоматически.

ЧЕСТНО О КАЧЕСТВЕ ПЕРЕВОДА
----------------------------
Это машинный перевод, не идеальный. Обычный режим (Argos) даёт быстрый,
но иногда корявый текст. LLM-доработка исправляет часть грамматических
ошибок (согласование падежей/рода), но не панацея — редкие слова обе модели
иногда путают. Для важных модов рекомендуется вычитать перевод вручную.
